# Выявленные дефекты и исправления

## 1. Обработка значений None
- **Функции**: Все методы класса
- **Проблема**: Необработанные исключения при передаче None
- **Исправлено**: Добавлена проверка типа с вызовом TypeError
- **Серьезность**: Высокая

## 2. Удаление символов без учета регистра
- **Функция**: delete_symbol
- **Проблема**: Регистрозависимое удаление символов
- **Пример**: delete_symbol("SkyPro", "s") → "kyPro" (должно быть "SkyPro")
- **Решение**: Использование регистрозависимой замены
- **Статус**: Исправлено

## 3. Проверка пустых строк
- **Функция**: is_empty
- **Проблема**: Строки с пробелами считались непустыми
- **Пример**: is_empty("   ") → False (должно быть True)
- **Решение**: Использование strip() вместо проверки длины
- **Статус**: Исправлено

## 4. Преобразование списка с None
- **Функция**: list_to_string
- **Проблема**: Ошибка при наличии None в списке
- **Решение**: Преобразование элементов в строку через map(str)
- **Статус**: Исправлено
